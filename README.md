## 📖 About

**PinGenie AI** is a client-side tool designed for bloggers and content creators to streamline their Pinterest marketing strategy. By leveraging powerful Large Language Models (LLMs), this tool analyzes your blog posts and generates premium Pinterest concepts, including:

*   **Visual Prompts:** Detailed descriptions to generate images (using Midjourney, DALL-E, etc.).
*   **Viral Titles:** Catchy headlines optimized for clicks.
*   **SEO Tags:** Relevant keywords to boost visibility.

## 🚀 Features

*   **Multi-Model Support:** Choose your preferred AI intelligence.
*   **Instant Analysis:** Paste your blog content and get results in seconds.
*   **Privacy First:** Your API keys and data are processed locally in your browser.
*   **User-Friendly Interface:** Clean design for a seamless workflow.

## 🧠 Supported AI Models

Select the intelligence that works best for you:

1.  **Gemini** (Google DeepMind)
2.  **GPT-4o** (OpenAI)
3.  **Llama 3.3** (Powered by GroqCloud)

## 🔒 Privacy & Security

**Your security is our priority.**

*   **Local Processing:** This application runs entirely in your browser (Client-Side).
*   **No Server Storage:** We do not have a backend server. Your API Key is **never** sent to us or stored on our servers. It is strictly used to communicate directly from your browser to the AI provider (Groq, OpenAI, or Google).

## 🛠️ How to Use

1.  **Visit the Website:** Go to [PinGenie AI](https://anees079.github.io/pingenie-ai/).
2.  **Select Intelligence:** Choose your desired AI provider (e.g., Llama 3.3 on Groq).
3.  **Enter API Key:** Input your API key (e.g., starts with `gsk_` for Groq).
4.  **Input Details:**
    *   **Blog Title:** (e.g., "10 Tips for Minimalist Living")
    *   **Content:** Paste your full blog post or a detailed summary.
5.  **Generate:** Click the button to receive your Pin prompts and SEO data.

## 💻 Installation (Run Locally)

If you wish to run this project on your local machine:

1.  Clone the repository:
    ```bash
    git clone https://github.com/anees079/pingenie-ai.git
    ```
2.  Navigate to the project folder:
    ```bash
    cd pingenie-ai
    ```
3.  Open `index.html` in your web browser.
